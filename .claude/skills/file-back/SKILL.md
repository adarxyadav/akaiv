# File back

A worked-out answer that lives only in chat evaporates — file it, and it compounds like a study.

1. **Find the owner.** An existing doc already owns this territory → the answer extends that doc. Owner missing → new doc under docs/, plus its one line in the index.
2. **Write it to be read cold:** prose a newcomer could follow with this conversation gone, linked to the docs it touches.
3. **One owner per fact.** When the answer moves a fact another doc states, edit the owner and grep the old value — every extra hit becomes a link to the owner.
4. **Log line:** date — what was filed — why it was worth keeping.

Done when the doc exists, the index points at it, and the grep for any moved fact comes back clean.
