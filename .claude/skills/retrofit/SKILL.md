# Retrofit

Runs from the stone, against a live project. The retrofit clause in Whetstone's COMPASS starter-template bullet is the contract; this runbook is the sequence two retrofits proved (SME24, EFICOR):

1. **Integrate, never duplicate.** The rules go into the project's existing agent entry file (CLAUDE.md, AGENTS.md, or equivalent) — a second doc root feeds the very drift this cures. A stale parallel doc system found along the way gets folded into the live one and deleted, on the human's explicit say-so.
2. **Seed the queue by mining.** The project's own docs hold the buried open items — "not done" fragments, page TODOs, features named twice and built never. Each becomes a ticket in the project's queue.
3. **Anchor the sync contract on the retrofit date,** written right beside the fork rules — a pre-existing log can't be trusted to hold a fixed line one.
4. **The client's surface stays theirs.** When the client keeps a success-criteria doc, it remains the client-facing surface; tickets tag its rows.
5. **Copy .claude/skills/** so the toolbox travels.

Done when the human confirms the seeded queue is the complete remaining scope — the check both retrofits passed.
