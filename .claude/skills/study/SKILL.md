# Study

Anything we learn from gets studied first, so the lesson outlives the borrowing.

1. **Ticket first,** via /ticket; the check is the rule-7 quiz.
2. **Save the source verbatim** to docs/raw/ — read, never edited. An article saves whole, images included. A repo pins its commit hash as the verbatim anchor and saves the files actually read, chosen so the links between saved files resolve.
3. **Distill** to docs/<name>-study.md in the house shape: what it is; ideas worth stealing; what it changed for us, dated; related studies linked both ways. Keep adoptions separate from recorded-not-adopted — a rejection written down is a decision kept.
4. **Adoptions land where they execute** — a rule line, a skill, a script — with the study holding the why and the link.
5. **Index line, log line,** then /close runs the quiz.

Done when raw, study, index, and log all exist and the quiz sits queued with the human.
